# Usage:


`ruby
> Κοτόπουλο = Κλάση.νέο
=> #<Class:0x007fbef59196a8> 
`

`ruby
> Κοτόπουλο.αποστολή :επιθεώρηση
=> "#<Class:0x007fbef59196a8>" 
`

`ruby
> Κοτόπουλο.μέθοδος :νέο
=> #<Method: Class(Class)#νέο(new)> 
`
 
`ruby
> Κοτόπουλο.κλάση.όνομα
=> "Class"
`

`ruby
> Κοτόπουλο.είναι? Κλάση
=> true 
`

`ruby
:σύμβολο.είναι? Σύμβολο
=> true 
`

`ruby
> [:ένα, :δύο, :τρία].συμπεριλαμβάνεί? :ένα
=> true 
`

`ruby
> Σειρά.νέο("φου μπάρ")
=> "φου μπάρ"
`

`ruby
> Παράταξη.νέο([:α, :β, :γ])
=> [:α, :β, :γ] 
`

`ruby
> {α: :β}.επιθεώρηση
=> "{:α=>:β}" 
`

`ruby
> Μίγμα.πρόγονοι
